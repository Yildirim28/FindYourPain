# FindYourPain 🎯
> **UIU Tentative Exam Routine Locator** — A sleek, minimalistic, and futuristic open-source web application designed to help UIU students instantly discover their exam destiny.

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen?style=for-the-badge)](https://yildirim28.github.io/FindYourPain/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg?style=for-the-badge)](LICENSE)

---

## ⚡ Features

- 🌌 **Futuristic Cyber Aesthetic**: Features beautiful background floating orbs, frosted glass panels (glassmorphism), and neon text shadows.
- 💥 **Interactive Glitch Header**: Hover-sensitive digital glitch title effect.
- 🔍 **Multi-Course Search**: Instantly look up multiple course codes at the same time by separating them with commas (e.g., `CSE 3711, CSE 4509, MATH 201`).
- 💡 **Smart Search Hints**: Auto-suggestion dropdown appears as you type. Clicking any course code auto-completes it and sets up a comma for the next entry!
- 📅 **Chronological Timetables**: Combines searched exams, filters duplicates, and displays them sorted by exam day and time slot.
- 🌊 **Circular Radar Wave**: Press the **Refresh** button to experience a gorgeous sonar ripple wave radiating directly from your click coordinate while clearing the search.
- 📱 **Fully Responsive Layout**: Fits flawlessly on all screens—from smartphones to widescreen desktop displays.
- 📁 **Complete Course Library**: Open the "All Courses" modal to see a scrollable list of every course exam scheduled in the database.

## 💻 Running Locally

Since this app is built purely using modern Vanilla Web Stack (**HTML5**, **CSS3**, **ES6 JavaScript**), it has zero dependencies and requires no server installation!

1. Clone this repository to your local system:
   ```bash
   git clone https://github.com/Yildirim28/FindYourPain.git
   ```
2. Double-click the `index.html` file to open it instantly in any modern web browser (Chrome, Edge, Firefox, Safari).
